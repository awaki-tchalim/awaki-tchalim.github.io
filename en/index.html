Portfolio

GSAPP, the emergency relief management application

GSAPP is the emergency management application of the ambulance and VSL company Togo assistance. The application has undergone many changes influenced by the evolution of the activities, needs and offers of the company. Thereby :
The very first version is developed in Java. (2014-2016) I developed it from A to Z as a Windows Java Swing application (Java desktop application) with the Netbeans IDE and MySql for database. I integrated JasperReports for reporting.

With the development of activities and the establishment of other agencies in other cities and in order to make subscribers' information available in real time, I opted for a temporary solution while waiting for the web version, to deploy the application previously available only in the local network on a remote server. I thus obtained a server under Debian 9 on the Google cloud platform and in a short time and temporarily put the application online through the Webswing web server

The working conditions, the constraints that arose with the expansion of activities, the search for more fluidity and a better user experience motivated my decision to take over GSAPP entirely as a web application. Hence version 2 with MySql, PHP, Laravel at the back end (sanctum for authentication and spatie for permissions and access rights), at the front end, Vue.js with laravel-mix and the Element UI component kit and npm as a package manager, PWA (Progressive Web Application), Xlsx for the export of data in Ms. Excell, vue-i18n (internalization),.

The fluidity of the application and its functionalities, especially the electronic medical log, plays a big role in the various interventions of Togo assistance and helps to save lives in the event of a disaster (traffic accident, work accident or illness)
